### Mục tiêu tuần 5:
* Hiểu lý thuyết và thực hành Storage Gateway, biết cách tạo File Share và kết nối với máy On-premise.
* Hiểu cách sử dụng CloudFront kết hợp với S3 để triển khai website tĩnh (static web).
* Biết cách cài đặt AWS CLI và thực hành VMware Import/Export để đưa máy ảo lên EC2.
* Hiểu lý thuyết và thực hành về AWS Security Hub để giám sát tình trạng bảo mật.
* Hiểu lý thuyết và thực hành về Lambda Function - dịch vụ serverless computing.
