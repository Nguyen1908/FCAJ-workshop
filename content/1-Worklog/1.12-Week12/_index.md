### Week 12 Results:
* **Foundational knowledge:** Organized an overview of how AWS networking components work together - VPC as a private network boundary, IGW/NAT providing internet connectivity for public/private subnets, and SG/NACLs controlling traffic at the instance and subnet levels.
* **CI/CD:** Reinforced the full CI/CD workflow from source code (GitLab/GitHub) → build/test (CodeBuild) → automated deployment (CodePipeline) onto different platforms such as ECS, EKS, and Lambda (Serverless).
* **SNS:** Solidified understanding of SNS as a publish/subscribe messaging service used for sending notifications and connecting components in event-driven architectures.
* **Storage and databases:** Consolidated the differences and use cases of storage services (S3, Storage Gateway) and databases (RDS for relational, DynamoDB for NoSQL), along with backup strategies using AWS Backup.
* **Overview:** Reviewed the entire 11-week journey, from getting started with an AWS account and basic networking infrastructure, to security (IAM, KMS), databases, and CI/CD and infrastructure as code (CDK). Identified areas that need further practice to prepare for upcoming content.
