This work journal records a 12-week AWS learning and practice program. During this time, I learned AWS concepts and services from basic to advanced, completed hands-on labs, and gained practical experience managing cloud infrastructure.
 
**Week 1:** [Attended kickoff, connected with FCAJ members, and learned the internship unit's rules](1.1-week1/)
 
**Week 2:** [Created and configured an AWS account, resolved errors, and learned about AWS Support and Budget](1.2-week2/)
 
**Week 3:** [Learned basic AWS networking (VPC, Subnet, IGW, NAT), EC2, Site to Site VPN, and CloudWatch](1.3-week3/)
 
**Week 4:** [Learned CloudFormation, connecting to RDGW via RDP, VPC Peering, Transit Gateway, and AWS Backup](1.4-week4/)
 
**Week 5:** [Learned Storage Gateway, CloudFront with S3, VMware Import/Export, Security Hub, and Lambda Function](1.5-week5/)
 
**Week 6:** [Learned resource management with Tags/Resource Groups, IAM, Permission Boundary, and encryption with KMS](1.6-week6/)
 
**Week 7:** [Learned Role Condition, Amazon RDS, AWS Glue/Athena, and Amazon DynamoDB](1.7-week7/)
 
**Week 8:** [Learned QuickSight, AWS SNS, AWS S3 and S3 security, and cost governance with IAM](1.8-week8/)
 
**Week 9:** [Learned CI/CD Pipeline on Amazon ECS, integrating GitLab/GitHub with CodeBuild, EKS Blueprints, and Serverless CI/CD](1.9-week9/)
 
**Week 10:** [Learned CI/CD on the EKS Workshop, AWS RDS PostgreSQL, Secrets Manager, Docker, and migrating to Microservices](1.10-week10/)
 
**Week 11:** [Learned basic and advanced AWS CDK, Systems Manager, Session Manager, and database migration](1.11-week11/)
 
**Week 12:** [Consolidated and organized all knowledge learned: foundational infrastructure, CI/CD, SNS, storage, and databases](1.12-week12/)
